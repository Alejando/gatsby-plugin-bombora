# gatsby-plugin-bombora

Easily add Bombora Id to your Gatsby site.

## Install
`yarn add gatsby-plugin-bombora`

## How to use

```javascript
// In your gatsby-config.js
plugins: [
  {
    resolve: `gatsby-plugin-bombora`,
    options: {
      bomboraEid: 'Bombora eid here',
      bomboraCid: 'Bombora cid here'
    },
  },
]
```
